# Arduino-Negative-Pressure-Gauge
Arduinoで作った真空ゲージ

手作り真空ポンプの性能を確かめたくて、デジタル真空ゲージを作ってみました。<br>
使ったセンサー(MIS-2500-015V)の性能ですが、最大-1000mBar(-0.1MPa)まで計ることができます。<br>
<br>
